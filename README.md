#  Malaria Incidence Trends in Africa (2000–2024)

## Overview

This project presents a comprehensive analysis of malaria incidence trends across East African countries using World Bank World Development Indicators (WDI) data. The goal is to uncover patterns, regional disparities, and long-term trends in malaria burden to support public health awareness and data-driven decision-making.

An interactive Power BI dashboard was developed to visualize country-level and regional insights, enabling users to explore malaria trends over time.


## Objectives

* Analyze malaria incidence trends from 2000 to 2024
* Compare malaria burden across East African countries
* Identify high-risk regions and temporal patterns
* Support public health insights through data visualization


## Tools & Technologies

* **Power BI** – Data visualization and dashboard development
* **Microsoft Excel** – Data cleaning and preprocessing
* **World Bank WDI Dataset** – Source of malaria incidence data


##  Dashboard Features

* **Time-Series Analysis**: Track malaria incidence trends over time
* **Country Comparison**: Compare incidence rates across East African countries
* **Interactive Filters**: Dynamic filtering by country and year
* **KPI Indicators**: Key metrics such as latest incidence rates and regional averages
* **Geospatial Visualization**: Map-based view of malaria burden distribution


## Key Insights

* Kenya shows a **consistent decline** in malaria incidence over time
* Uganda maintains a **relatively higher malaria burden** compared to neighboring countries
* Significant **regional disparities** exist across East Africa
* Progress is evident, but malaria remains a **major public health concern**


## Challenges

* Incomplete or inconsistent reporting for some countries and years
* Variability in data availability across the region
* Limited granularity (national-level data only)


## Recommendations

* Strengthen malaria surveillance systems in high-burden regions
* Improve data collection consistency across countries
* Leverage data-driven approaches for targeted interventions
* Use similar dashboards to support public health policy decisions



## Project Impact

This project demonstrates how data analytics can be applied in a public health context to:

* Monitor disease trends
* Identify high-risk populations
* Support evidence-based decision-making

It reflects the role of data science in **epidemiology and health research**, aligning with organizations such as KEMRI.
