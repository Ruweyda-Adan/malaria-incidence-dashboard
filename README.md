# Malaria Incidence Analysis Dashboard 

## Overview
This project analyzes malaria incidence trends across East African countries from 2000 to the latest available year using World Bank data.

## Tools Used
- Power BI
- Excel (data cleaning)
- World Bank WDI dataset

## Key Insights
- Kenya shows a declining trend in malaria incidence over time
- East African countries show varying levels of malaria burden
- Comparative analysis highlights regional disparities

## Features
- Time-series analysis (2000–latest year)
- Country comparison dashboard
- Interactive filters by country and year
- KPI indicators for key metrics
